# 리스트 정렬 sort
기본 값은 오름차순 정렬, reverse 사용으로 내림차순 정렬  
ex)   
n = [1, 2, 5, 3, 4]  
n.sort()  
print(n) -> [1, 2, 3, 4, 5]  

n.sort(reverse=True)  
print(n) -> [5, 4, 3, 2, 1]

key값으로 정렬도 가능  
ex)  
s = ["생각한다", "나", "그리고", "너는"]  
s.sort(key=len) -> 문자 길이로 정렬  
print(s) -> ["나", "너는", "그리고", "생각한다"]
